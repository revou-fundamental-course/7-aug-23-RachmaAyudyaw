Luas Segitiga

Rumus Luas segitiga yaitu:

L = 1/2 x a x t

Di mana:

L = Luas

a = Panjang Atas

t = Tinggi

Keliling Segitiga

Rumus keliling segitiga yaitu:

K= S1 + S2 + S3

Di mana:

K = Keliling

S1 = Sisi A

S2 = Sisi B

S3 = Sisi C
